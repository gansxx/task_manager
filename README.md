## Quick Start
Task Manager starts working as soon as you enable it(**Zero Config**):

- Create a task and the plugin automatically appends `@start(YYYY-MM-DD)`.
- Complete a task and  the plugin automatically appends `@done(YYYY-MM-DD)`.
- **Right click** to Mark task priority with `@priority(urgent|high|medium|low)` and manage it from the editor menu or commands.
